## 📌 Overview
High-fidelity deepfakes and audio-based fraud (vishing) are becoming increasingly difficult to distinguish from reality. **The Digital Audio Notary** is a web-based audio forensic toolkit designed to provide a definitive "Trust Audit." By analyzing uploaded audio files, this tool extracts deep acoustic features to accurately classify whether a voice is human or synthetically generated by AI.

🌐 **Live Demo:** [Click here to view the live web app](https://hugging-u1-forensic-tool-kit.hf.space)

## 🚀 Key Features
* **Deep Feature Extraction:** Utilizes `librosa` to analyze microscopic digital artifacts and phase alignments that human vocal cords cannot naturally produce.
* **Format Flexibility:** Supports standard formats (`.wav`, `.flac`) as well as mobile audio formats (`.mp3`, `.m4a`) via an integrated `ffmpeg` pipeline.
* **Ephemeral Processing:** Built with a secure backend that temporarily stores files in an `/uploads` directory, processes the forensic audit, and manages the data securely.
* **Responsive UI:** Features a sleek, dark-mode graphical interface with drag-and-drop functionality built directly into the Flask application.

## 🛠️ Tech Stack
* **Backend:** Python 3.9, Flask
* **Audio Forensics:** Librosa, NumPy
* **System Tools:** FFmpeg (for mobile audio decoding)
* **Frontend:** HTML5, CSS3 (Custom Dark Theme), Vanilla JavaScript
* **DevOps / Deployment:** Docker, Git, Hugging Face Spaces

## 🧠 Known Limitations: The "Replay Attack"
Currently, this model is trained to detect *digital* fingerprints left behind by AI synthesis. 
If an AI-generated voice is played out of a physical speaker and re-recorded with a microphone (a **Replay Attack**), the physical acoustic environment (speaker distortion, room reverb, microphone hardware) acts as a filter, scrubbing away the pristine digital AI markers. For the highest accuracy, please upload **original digital files** rather than physical re-recordings.
